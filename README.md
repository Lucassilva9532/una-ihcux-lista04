# una-ihcux-lista04
O bloco try-catch é usado para capturar e tratar erros (exceções) em tempo de execução, evitando que a aplicação seja interrompida.
try: contém o código suscetível a erro
catch: captura e trata a exceção
finally: executa sempre (opcional)
